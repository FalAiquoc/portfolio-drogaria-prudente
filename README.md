# Drogaria Prudente — Site Institucional
Site desenvolvido como projeto de portfólio por Diogo Falcão (FalAiquoc)

🏪 Sobre o Negócio
A Drogaria Prudente atua no bairro de Lagoa Nova em Natal/RN, sendo referência no atendimento farmacêutico da região. A loja conta com medicamentos de marca, genéricos acessíveis, além de uma ampla seção de dermocosméticos e suplementos.

🛠️ Tecnologias
- Vite + React + TypeScript
- Tailwind CSS
- Lucide React

📐 Referências de Design
- https://github.com/adrianhajdin/project_hoobank — Referência para layout modular moderno com Tailwind CSS.
- https://github.com/WebDevSimplified/react-shopping-cart — Estrutura conceitual de vitrine e cards de produtos de forma dinâmica.
